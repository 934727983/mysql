# ST\_trajAttrsAsText {#reference_dxk_ddq_3gb .reference}

获得作为文本类型的轨迹属性数组。

## 语法 {#section_og1_4hn_qfb .section}

```
text[]  st_trajAttrsAsText(trajectory traj, text attr_name);
```

## 参数 {#section_cxv_qhn_qfb .section}

|参数名称|描述|
|----|--|
|traj|轨迹对象。|
|a​ttr\_name|属性名称。|
