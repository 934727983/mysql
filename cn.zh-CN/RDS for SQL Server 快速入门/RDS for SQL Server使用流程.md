# RDS for SQL Server使用流程 {#concept_fzw_stz_vdb .concept}

快速入门旨在介绍如何创建RDS实例、进行基本设置以及连接实例数据库，使用户能够了解从购买RDS实例到开始使用实例的流程。

## 快速入门流程图 { .section}

若您初次使用阿里云RDS，请先了解[RDS使用限制](intl.zh-CN/RDS for SQL Server 快速入门/使用限制.md#)。
